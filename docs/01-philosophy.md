# 1HP — Philosophy & Language System

## Core Definition

**1HP — One Human Powered**

Every movement powered by a human body. No engines, no motors, no optimisation obsession. Just humans moving themselves through space.

---

## Philosophy (Non-Negotiable)

### What We Believe

| Belief | Meaning |
|--------|---------|
| Movement is not a competition by default | Participation has value without ranking |
| The body is not a machine | It is not something to be exploited or optimised |
| Consistency > extremes | Long-term patterns matter more than peaks |
| Humans move for many reasons | Speed is one reason among many |
| Walking counts | All human-powered movement is valid |
| Rest counts | Absence of activity is part of the system |
| Aging counts | Bodies change; the system should reflect that |
| Returning counts | Coming back after absence is meaningful |

### What We Reject

- "No pain, no gain"
- Hustle culture applied to the body
- Ranking humans by output
- Reducing movement to numbers alone
- Turning participation into pressure

---

## Language Rules

### Use These Words

| Word | Why |
|------|-----|
| human-powered | Core identity |
| state | Current condition, not judgment |
| participation | Presence matters |
| continuity | Long arc over isolated moments |
| balance | Cross-activity equilibrium |
| baseline | Personal reference, not comparison |
| effort | What humans give |
| movement | Universal term for all activity |

### Avoid These Words

| Word | Why |
|------|-----|
| grind | Glorifies suffering |
| hustle | Startup culture, not bodies |
| crush | Violence metaphor |
| peak | Implies decline |
| PR | Performance obsession |
| beast | Dehumanizing |
| optimisation | Bodies are not machines |
| domination | Competition framing |

---

## Tone

- **Calm** — No urgency, no pressure
- **Adult** — Respect intelligence
- **Respectful** — No condescension
- **Unimpressed by extremes** — Consistency over heroics

---

## Tagline System

### Primary

> **1HP — One Human Powered**

### Supporting (use sparingly)

- Movement, by humans
- Powered by you
- Human effort, considered
- Where movement lives
- Understanding movement, not chasing it

### Runstate Descriptor

> Runstate — understanding your body across every human-powered effort

**Rule**: Never shout taglines. They are identifiers, not calls to action.

---

## Boundaries (Hard Lines)

1HP must **NEVER**:

- Shame inactivity
- Celebrate exhaustion
- Glorify suffering
- Rank humans implicitly or explicitly
- Suggest worth through output

If a feature, line of copy, or design choice violates this philosophy, it is wrong — even if it "works".

---

## The Litmus Test

Every decision must pass:

> "Would this still make sense to a human 20 years from now, moving for reasons we can't predict?"

If not, do not ship.
